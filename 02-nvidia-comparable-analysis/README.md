# NVIDIA Corporation — Comparable Company Analysis

![Project Banner](images/hero_banner.png)

[![Status](https://img.shields.io/badge/Status-Coming%20Soon-lightgrey)](#)
[![Ticker](https://img.shields.io/badge/NASDAQ-NVDA-1F4E78)](#)

A comparable company analysis of NVIDIA Corporation benchmarked against semiconductor and AI infrastructure peers. The analysis covers trading multiples, growth-adjusted valuations, relative positioning, and an implied valuation range derived from the peer set.

## Methodology

**Comparable Company Analysis (Trading Comps)**

## Planned Deliverables

- Peer universe selection and screening criteria
- Trading multiples benchmarking (EV/Revenue, EV/EBITDA, P/E, PEG)
- Growth-adjusted valuation analysis
- Implied valuation range and football field chart
- Executive summary with investment perspective

## Skills Demonstrated

`Financial Modeling` `Comparable Analysis` `Equity Valuation` `Data Visualization` `Excel`
