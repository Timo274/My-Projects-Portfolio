# Adobe–Figma — M&A Case Study

![Project Banner](images/hero_banner.png)

[![Status](https://img.shields.io/badge/Status-Coming%20Soon-lightgrey)](#)
[![Ticker](https://img.shields.io/badge/NASDAQ-ADBE-1F4E78)](#)

An M&A case study analyzing Adobe's proposed $20B acquisition of Figma. The project covers strategic rationale, deal structure, accretion/dilution analysis, synergy assumptions, regulatory considerations, and a recommendation on whether the deal creates shareholder value.

## Methodology

**M&A Analysis / Accretion-Dilution**

## Planned Deliverables

- Deal overview and strategic rationale
- Transaction structure and financing analysis
- Accretion / dilution analysis
- Synergy quantification and sensitivity
- Regulatory risk assessment and recommendation

## Skills Demonstrated

`M&A Analysis` `Accretion/Dilution` `Strategic Analysis` `Financial Modeling` `Excel`
