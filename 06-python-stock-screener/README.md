# Quantitative Stock Screener



[![Status](https://img.shields.io/badge/Status-Coming%20Soon-lightgrey)](#)


A Python-based quantitative stock screener that filters US equities using fundamental, valuation, and momentum criteria. The tool retrieves live market data, applies customizable screening rules, and outputs ranked results with key financial metrics.

## Methodology

**Quantitative Screening / Factor-Based Analysis**

## Planned Deliverables

- Configurable multi-factor screening engine
- Fundamental and valuation metric computation
- Momentum and technical filters
- Ranked output with exportable results
- Documentation and usage guide

## Skills Demonstrated

`Python` `Quantitative Finance` `Data Engineering` `API Integration` `Pandas`
